# PRD: 정보시스템 자산관리 솔루션

## 목표
공공기관 부서 내 서버/네트워크/정보보호시스템 자산의 등록부터 폐기까지 전 생명주기를 통합 관리하고, 랙실장도/네트워크 포트맵/MDF·TPS 배선 현황을 시각적으로 제공하는 웹 기반 솔루션

## 사용자
- 정보운영과/정보보호과 IT 인프라 관리 담당자
- 전산실/서버실 운영 관리자
- 네트워크/보안 장비 담당자

## 핵심 기능
1. **자산 CRUD** — 서버, 네트워크, 정보보호, 스토리지 장비 등록/조회/수정/삭제
2. **랙 실장도** — 랙별 장비 배치 시각화 (유형별 색상, U 단위, 호버 상세)
3. **네트워크 포트맵** — 스위치/장비별 포트 그리드, 상태/VLAN/연결 추적
4. **MDF/TPS 배선관리** — 건물→층→배선반→110블록 페어 단위 관리/시각화
5. **대시보드** — 자산 현황, 상태 분포, EoS/보증만료 경고, 데이터 품질 점수
6. **다중 IP 관리** — 자산별 인터페이스/IP/서브넷/게이트웨이 관리
7. **엑셀 일괄등록/내보내기** — 양식 다운로드 → 작성 → 업로드 (검증 오류 표시)
8. **사진 업로드** — 자산별 사진 첨부 + AI 분석 어댑터 아키텍처
9. **변경 이력** — 등록/수정/삭제 감사 로그 (누가/언제/무엇을)
10. **QR코드 라벨** — 자산태그 기반 QR 생성/인쇄
11. **로그인/인증** — RBAC (admin/user/viewer), HMAC 세션
12. **동적 커스텀 필드** — 사용자가 UI에서 자산 속성 자유 추가

## MVP 제외 사항
- 실시간 SNMP/에이전트 모니터링 (Zabbix/PRTG 영역)
- 자동 네트워크 탐색 (auto-discovery)
- 전력/냉각/환경 모니터링 (IoT 센서 필요)
- 3D 시각화 (과도한 프론트엔드 복잡도)
- 소프트웨어 라이선스 관리 (별도 시스템)
